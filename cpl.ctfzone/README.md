# cpl.ctfzone

## Overview

This directory stores local CTFZone challenge material and solve notes. Each
challenge gets its own subdirectory with the original artifact, a README, and a
small PoC script for reproducing the solve path.

## Challenge Folders

- `chained`

## Recommended Workflow

1. Open the challenge README.
2. Inspect the local artifact in the challenge folder.
3. Run the PoC script to reproduce the analysis.
4. Compare the intermediate output with the final flag candidate.
